# 🍕 Pizza Sales Dashboard (Power BI + SQL)

This project is a **Power BI dashboard** built using **SQL and data visualization techniques**
to analyze pizza sales data and generate business insights.

## 📊 Dashboard Pages

### 🏠 Home Page
- Key KPIs:
  - Total Revenue
  - Average Order Value
  - Total Pizzas Sold
  - Total Orders
  - Average Pizzas per Order
- Daily & Monthly sales trends
- Sales breakdown by:
  - Pizza Category
  - Pizza Size
- Insights on busiest days and times

### ⭐ Best/Worst Sellers Page
- Identifies:
  - Top-selling pizzas
  - Lowest-selling pizzas
- Helps in product performance analysis

## 🔄 Navigation
- Implemented using **Page Navigator**
- Easy switching between:
  - Home
  - Best/Worst Sellers

## 🛠 Tools & Technologies
- **SQL** → Data extraction & querying  
- **Power BI** → Dashboard & visualization  
- **DAX** → Calculated measures  

## 🎯 Project Objective
- Analyze sales performance  
- Find trends and patterns  
- Identify best & worst products  
- Support data-driven decisions
